## Animation Optimization

- Size changes are costly
- Some properties are GPU accelearated
  - `transform`
  - `filter`
  - `opacity`
- Using `will-change`
  - Last resort
  - Do not apply to too many elements

> Premature optimization? It's relatively easy to write it correctly first time

</Slide>
