A short history of the different iterations of this website

## 2018-2018
  
> March 11, 2018 was apparently the first ever version of my website. This was a little starter blog and was the first Angular app I ever built - a single page app with three components in a "posts" folder.
  
<iframe src="https://2018.archive.nabeelvalley.co.za"></iframe>

## 2019-2020

> The first "real" version of my website with the main goal of making my notes more easily accessible. Built as a regular-old react app
  
<iframe src="https://2019.archive.nabeelvalley.co.za"></iframe>

## 2020-2021
  
> A rewrite of the 2019 site using the same design and reusing a lot of the components but using Gatsby. The focus of this build was to make the site faster overall and it did a pretty good job at that
  
<iframe src="https://2021.archive.nabeelvalley.co.za"></iframe>

## 2021-2022

  
> The Gatsby site started to struggle to keep up with amount of content and the site started to take ages to build, particularly running into out of memory issues when building in the cloud and was a hassle to maintain. This version was built using Eleventy with a slight redesign to simplify the content structure. I also moved the photography to a different site during this period so this overall design placed a lot less emphasis on imagery compared to previous versions

<iframe src="https://2022.archive.nabeelvalley.co.za"></iframe>

## 2022-2023

  
> I was reaching a point where I wanted to try to integrate some more complex functionality into my site as well as make it possible for me to build self-contained applet type things in my site. I wanted a framework that would enable me to play around in that space. This version of the site was written with Astro and had a few small experiments such as a small JavaScript playground that was all built within the app

<iframe src="https://2023.archive.nabeelvalley.co.za"></iframe>

## 2023-2026
  
> The site you're looking at. A complete redesign, still using Astro, but now with MDX. This introduced a bunch of new ideas around how components could work as well as introduced me to some interesting ideas around how to develop more complex content for the site like photo galleries and presentations
  
<iframe src="https://2026.archive.nabeelvalley.co.za"></iframe>

## 2026-Present

The current, work-in-progress, website that you're looking at
